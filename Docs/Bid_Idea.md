💡 PHASE 5: Big Idea + Corporate Identity + Design System

ส่วนที่ 1: Big Idea — 3 แนวคิดให้เลือก

🎬 BIG IDEA #1: "FRAME OF THAILAND"
แนวคิด: ทุกเรื่องราวมีกรอบ — กรอบภาพยนตร์ไทยที่ยิ่งใหญ่

Core Concept: ภาพยนตร์คือ "กรอบ" ที่บันทึกเรื่องราวของชาติ ContentThailand คือกรอบใหญ่ที่รวมทุกกรอบเล็กๆ ของภาพยนตร์ไทยไว้ด้วยกัน

Visual Direction:

ใช้ "Film Frame" เป็น design motif หลัก — ทุก element ถูกกรอบด้วย cinematic frame
สี: Deep Midnight Blue + Gold + Pure White — สื่อถึงความยิ่งใหญ่ คลาสสิก และทันสมัย
Typography: Modern Thai serif + clean sans-serif English
Motion: Parallax scrolling เหมือนกล้องเคลื่อน

🔥 ทำไมถึง WOW:

Visual metaphor ชัด — กรอบภาพยนตร์ทุกคนรู้จักทันที
สร้าง brand memory ได้แข็งแกร่ง — เห็นครั้งแรกก็รู้ว่าคือเว็บภาพยนตร์
กรรมการเห็นความ creative ที่เชื่อมกับ core business

📣 ทำไมถึง Viral:

Share card ทุกหน้าออกมาในรูปแบบ "film frame" — Instagram-worthy ทันที
User แชร์ profile ตัวเองใน "กรอบภาพยนตร์" บน social media
Hashtag: #กรอบภาพยนตร์ไทย #FrameOfThailand

🏛️ สร้าง Brand Awareness อย่างไร:

ทุกครั้งที่เห็น frame สีทอง คนนึกถึง ContentThailand
โลโก้ใช้ film frame + ตัวอักษร CT ที่จดจำได้
สอดคล้องกับ mission ของกองภาพยนตร์ฯ อย่างสมบูรณ์

VISUAL TONE

  ✅ DO:
  - ภาพ still จากภาพยนตร์ คุณภาพสูง
  - Portrait ของบุคลากรที่เป็นระดับมืออาชีพ
  - Cinematic composition — rule of thirds
  - ภาพที่มี drama และ emotion
  - Overlay สี Midnight Blue แบบ subtle

  ❌ DON'T:
  - ภาพ stock photo ทั่วไป
  - ภาพถ่ายธรรมดาไม่มี art direction
  - สีสดจัด ขัดกับ palette หลัก
  - ภาพที่ไม่เกี่ยวกับภาพยนตร์หรือวัฒนธรรม

🎬 Motion & Animation Guidelines
ANIMATION PRINCIPLES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Timing:       Ease-in-out / 200-400ms
                (รวดเร็วแต่ไม่กระชาก)

  Loading:      Film strip horizontal scroll
                (แทน spinner ธรรมดา)

  Hover:        Scale 1.02 + Gold border appear
                (subtleและ elegant)

  Page Trans:   Fade + slight slide up
                (เหมือนฉากเปลี่ยนในหนัง)

  Skeleton:     Film-grain texture
                (ขณะโหลดข้อมูล)

  Scroll:       Parallax บน hero images
                (ความลึกแบบ cinematic)
