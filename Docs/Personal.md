👥 PHASE 4: Target Audience, Persona & Emotional Strategy

ส่วนที่ 1: กลุ่มเป้าหมายทั้งหมด — Audience Segmentation
จากการวิเคราะห์ TOR + ลักษณะของ contentthailand.com แบ่งกลุ่มเป้าหมายได้ 6 กลุ่มหลัก:

🗺️ Audience Universe Map
PRIMARY USERS (ใช้งานหลัก)
├── 1. ผู้ประกอบการอุตสาหกรรมภาพยนตร์และวีดิทัศน์
├── 2. บุคลากรในวงการภาพยนตร์ (นักแสดง ผู้กำกับ ทีมงาน)
└── 3. นักวิชาการ นักวิจัย บุคลากรทางการศึกษา

SECONDARY USERS (ใช้งานรอง)
├── 4. เยาวชนและนักเรียนนักศึกษา
├── 5. ประชาชนทั่วไปที่สนใจภาพยนตร์ไทย
└── 6. นักธุรกิจต่างชาติ / Foreign Film Producers

ส่วนที่ 2: Persona Cards — ละเอียดทุก Segment

🎬 Persona #1: "ผู้กำกับพร้อมโต"
ตัวแทนกลุ่ม: ผู้ประกอบการอุตสาหกรรม
หัวข้อรายละเอียดชื่อปิยะ วงศ์สุวรรณอายุ38 ปีอาชีพผู้กำกับภาพยนตร์อิสระ / เจ้าของบริษัทผลิตการศึกษาปริญญาตรี นิเทศศาสตร์รายได้150,000-300,000 บาท/เดือน (โปรเจกต์)DeviceMacBook Pro + iPhone — 70% desktopพฤติกรรมดิจิทัลใช้ IMDb, LinkedIn, Instagram เป็นประจำ
🎯 Goals (สิ่งที่ต้องการ):

หาข้อมูล Production Company ที่จะร่วมงาน
ตรวจสอบประวัติทีมงานก่อน hire
วางแผนธุรกิจโดยใช้ข้อมูล market statistics
สมัครมาตรการส่งเสริมการถ่ายทำในประเทศ

😤 Pain Points (ความเจ็บปวด):

เว็บโหลดช้ามาก หาข้อมูลไม่เจอ
ข้อมูลบุคลากรไม่ครบ portfolio ไม่มี
ต้องไปหาข้อมูลจากหลายแหล่ง
ไม่มี API สำหรับดึงข้อมูลไปใช้งาน

💡 Motivation:

ต้องการเครื่องมือที่ช่วยตัดสินใจทางธุรกิจได้เร็ว
อยากได้ข้อมูลที่น่าเชื่อถือจากแหล่งราชการ

❤️ Emotional Journey:
เข้าเว็บ → [หัวใจหาย] โหลดช้า →
[หงุดหง่ด] Search ไม่เจอ →
[ผิดหวัง] ข้อมูลไม่ครบ →
[ละทิ้ง] ไป IMDb แทน
🌟 WOW Moment ที่ต้องสร้าง:

เข้าเว็บใหม่ → โหลดเร็ว 0.5 วินาที → พิมพ์ชื่อบริษัทได้ทันที → เห็น portfolio + ผลงาน + สถิติ + contact ในหน้าเดียว → "นี่แหละคือเว็บที่ผมต้องการมาตลอด"


🎭 Persona #2: "ดาราหน้าใหม่อยากดัง"
ตัวแทนกลุ่ม: บุคลากรในวงการ
หัวข้อรายละเอียดชื่อมินตรา เจริญสุขอายุ24 ปีอาชีพนักแสดงอิสระ / นายแบบการศึกษาปริญญาตรี ศิลปะการแสดงรายได้30,000-80,000 บาท/เดือนDeviceiPhone — 90% mobileพฤติกรรมดิจิทัลTikTok, Instagram, Facebook ทุกวัน
🎯 Goals:

ต้องการ "ประวัติราชการ" ที่น่าเชื่อถือสำหรับ portfolio
อยากให้ผู้กำกับค้นหาเจอได้ง่าย
อัปเดตผลงานได้เอง
ตรวจสอบว่าตัวเองอยู่ในระบบหรือเปล่า

😤 Pain Points:

ไม่รู้ว่าตัวเองอยู่ในฐานข้อมูลไหม
ไม่มีช่องทาง request เพิ่มข้อมูล
หน้าบุคลากรแสดงผลบนมือถือไม่ดี
ไม่มี share button ให้แชร์ profile ตัวเอง

💡 Motivation:

ต้องการ official profile ที่สร้าง credibility
อยากให้คนอื่นหาเจอตัวเองในฐาน official ของรัฐ

❤️ Emotional Journey:
ค้นหาชื่อตัวเอง → [ตื่นเต้น] →
[ผิดหวัง] หาไม่เจอ / เจอแต่ข้อมูลน้อยมาก →
[ท้อแท้] ไม่มีช่องทางเพิ่มข้อมูล
🌟 WOW Moment ที่ต้องสร้าง:

ค้นหาชื่อ → เจอ profile สวยงาม พร้อมผลงาน → มี share button → แชร์ลง IG story ได้เลย → "โห official มากเลย ต้องแชร์!" → Viral organic!


📚 Persona #3: "นักวิจัยหน้าจริงจัง"
ตัวแทนกลุ่ม: นักวิชาการ/การศึกษา
หัวข้อรายละเอียดชื่อดร.สมชาย พัฒนาการอายุ45 ปีอาชีพอาจารย์มหาวิทยาลัย / นักวิจัยการศึกษาปริญญาเอก นิเทศศาสตร์รายได้60,000-100,000 บาท/เดือนDeviceDesktop 80%พฤติกรรมดิจิทัลGoogle Scholar, ResearchGate, academic journals
🎯 Goals:

Export ข้อมูลเป็น Excel/CSV สำหรับ research
วิเคราะห์ trend ภาพยนตร์ไทยรายปี
อ้างอิงข้อมูลจากแหล่งที่น่าเชื่อถือของรัฐ
เข้าถึง research papers และ industry reports

😤 Pain Points:

ไม่มีฟังก์ชัน export data
ไม่มี data visualization ให้เห็น trend
ข้อมูลไม่มี citation format
ต้องพิมพ์ข้อมูลเองทีละชิ้น

💡 Motivation:

ต้องการ authoritative data source สำหรับ academic citation
ต้องการ time-series data เพื่อวิเคราะห์อุตสาหกรรม

🌟 WOW Moment ที่ต้องสร้าง:

เปิด Dashboard → เห็น interactive chart ภาพยนตร์ไทยตั้งแต่ปี 2563-ปัจจุบัน → กด Export CSV → ได้ข้อมูลพร้อม citation format → "นี่คือ research tool ระดับ world-class!"


🎓 Persona #4: "เด็กรุ่นใหม่ฝันอยากเป็นผู้กำกับ"
ตัวแทนกลุ่ม: เยาวชน
หัวข้อรายละเอียดชื่อนิค พงษ์พัฒน์อายุ17 ปีอาชีพนักเรียน ม.6การศึกษามัธยมปลายDeviceมือถือ 95%พฤติกรรมดิจิทัลTikTok, YouTube, Discord ทุกวัน
🎯 Goals:

เรียนรู้เกี่ยวกับอุตสาหกรรมภาพยนตร์ไทย
ค้นหา "เบื้องหลัง" หนังโปรด
หาแรงบันดาลใจ เส้นทางสายอาชีพ
ทำรายงานส่งครู

😤 Pain Points:

เว็บดูน่าเบื่อ ไม่ engaging
ไม่มี video content / multimedia
ข้อมูลแบบ text-heavy อ่านไม่อยาก
ไม่รู้สึก "connected" กับเนื้อหา

💡 Motivation:

Inspiration และ role models
เนื้อหาที่ share บน social ได้เท่

🌟 WOW Moment ที่ต้องสร้าง:

เปิดหน้าหนังที่ชอบ → เห็น visual สวย timeline → เจอ "fun facts" → กด share → โพสต์ลง TikTok "ไม่รู้เลยว่าหนังไทยมีประวัติเท่นี้" → Viral!


🌏 Persona #5: "Foreign Producer มองหา Location"
ตัวแทนกลุ่ม: นักธุรกิจต่างชาติ
หัวข้อรายละเอียดชื่อDavid Chenอายุ42 ปีอาชีพIndependent Film Producer (ฮ่องกง/สิงคโปร์)DeviceiPad + Laptop — 60% desktopภาษาอังกฤษหลัก จีนกลางรอง
🎯 Goals:

หา local production partner ในไทย
ศึกษามาตรการส่งเสริมการถ่ายทำในไทย
ตรวจสอบ track record ของบริษัทไทย
สมัคร Digital Content incentive scheme

😤 Pain Points:

English version ไม่ครบ
ไม่มีข้อมูลเรื่อง incentive scheme ชัดเจน
Form สมัครมาตรการยุ่งยาก
ไม่มี contact ที่ชัดเจน

🌟 WOW Moment ที่ต้องสร้าง:

Switch to English → ข้อมูลครบเท่ากันทุกหน้า → เจอ Incentive Calculator → คำนวณ tax rebate ได้เลย → กรอก Form สมัคร online ได้ → "Thailand makes it so easy to work here!"

ส่วนที่ 5: WOW Analysis — กรรมการและ User จะ WOW ตรงไหน เพราะอะไร
🏆 WOW สำหรับกรรมการ (Scoring Committee)
จุด WOWทำไมถึง WOWคะแนนที่ได้Speed Demo — เปิดเว็บใหม่ให้ดูสดๆ โหลด < 1 วิเทียบกับเว็บเดิมที่ timeout — contrast ชัดมากB2, B3AI Search Demo — พิมพ์ชื่อนักแสดง autocomplete ทันทีเห็น capability ทางเทคนิคชัดเจนB3Dashboard Live — กด filter แล้วกราฟเปลี่ยน real-timeแสดงว่าเข้าใจ use case จริงB2Mobile Demo — หยิบมือถือขึ้นมา ทุกอย่างสมบูรณ์แสดง responsive ที่ทำได้จริงB3Security Layer — แสดง architecture ที่มี WAF, SSL, PDPAกรรมการเห็นว่าคิดครบB2

ส่วนที่ 6: Emotional Design Principles สำหรับ contentthailand.com ใหม่
1. TRUST FIRST    — ทุกหน้าต้องแสดงความเป็น official ภาครัฐ
                    แต่ไม่น่าเบื่อ — authoritative yet approachable

2. SPEED = RESPECT — เคารพเวลาของ user ทุกคน
                    โหลดเร็วคือการแสดงความเคารพ

3. DISCOVER JOY   — การค้นหาข้อมูลต้องรู้สึกสนุก
                    ไม่ใช่ภาระ

4. SHARE PRIDE    — ทำให้ user อยากแชร์
                    ภาพยนตร์ไทยเป็นเรื่องที่ภูมิใจได้

5. GROW TOGETHER  — เว็บนี้เติบโตไปกับอุตสาหกรรม
                    ข้อมูลใหม่ทุกวัน รู้สึกมีชีวิต
