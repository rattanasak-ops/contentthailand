# Team Roles Definition — ContentThailand Project
## ตำแหน่งทีมที่จำเป็นในการทำโครงการให้ออกมา WOW ที่สุด

---

## 1. Project Manager / Technical Lead (หัวหน้าโครงการ)

**JD (Job Description):**
- บริหารโครงการตาม Timeline 150 วัน, 4 Milestones
- ประสานงานระหว่างทีมพัฒนา, กรรมการ, และหน่วยงานราชการ
- ควบคุมคุณภาพ (QA), Risk Management, Change Request
- รายงานความคืบหน้ารายสัปดาห์ต่อ Steering Committee
- บริหาร Agile-Waterfall Hybrid workflow

**JS (Job Specification):**
- ปริญญาโทขึ้นไป สาขา IT, Computer Science, หรือ Management
- PMP / PSM / CSM certification จะพิจารณาเป็นพิเศษ
- ประสบการณ์ 15+ ปี ในการบริหารโครงการ IT ภาครัฐ
- เคยบริหารโครงการมูลค่า 5M+ THB อย่างน้อย 3 โครงการ

**Skills:**
- Project Management (Asana, Jira, Gantt Chart)
- Stakeholder Management — สื่อสารกับกรรมการราชการได้
- Risk & Change Management
- ISO/IEC 29110 knowledge
- Agile/Scrum methodology

---

## 2. UX/UI Designer (นักออกแบบ UX/UI)

**JD:**
- ออกแบบ UI ทั้ง Front-end (11+ หน้า) และ Back-end (CMS Dashboard)
- สร้าง Design System ตาม "Frame of Thailand" Cinematic Identity
- ออกแบบ Responsive (Mobile/Tablet/Desktop) ตามมาตรฐาน WCAG 2.1 AA
- สร้าง Figma Prototype สำหรับ UAT
- ออกแบบ Campaign visuals (Share Card, Quiz UI, Youth Challenge)

**JS:**
- ปริญญาตรี สาขา Design, Interactive Media, หรือเทียบเท่า
- ประสบการณ์ 5+ ปี ในงาน UX/UI Design
- Portfolio มีงาน Government / Enterprise website
- เข้าใจ WCAG 2.1 AA Accessibility Standards

**Skills:**
- Figma (Advanced) — Component, Auto Layout, Prototype
- Design System creation
- Accessibility design (color contrast, keyboard nav, screen reader)
- Motion design (Framer Motion concepts)
- Bilingual typography (Thai/English)
- Color theory — Dark theme design

---

## 3. Full-Stack Developer — Front-end Focus (นักพัฒนาส่วนหน้า)

**JD:**
- พัฒนา Front-end ด้วย Next.js 14 (App Router) + TypeScript + Tailwind CSS
- Implement Responsive Design, Animations (Framer Motion)
- สร้าง Bilingual system (TH/EN toggle)
- พัฒนา Search (Instant Search), Filtering, Pagination
- Optimize performance ให้ LCP ≤ 2.5s (ปัจจุบัน 21.41s)

**JS:**
- ปริญญาตรี สาขา Computer Science, IT, หรือเทียบเท่า
- ประสบการณ์ 5+ ปี ในงาน Web Development
- เชี่ยวชาญ React/Next.js ecosystem
- มีผลงาน website ที่ Lighthouse score 90+

**Skills:**
- Next.js 14 (App Router, SSR, ISR, RSC)
- TypeScript
- Tailwind CSS (advanced: custom theme, CSS variables)
- Framer Motion (scroll-reveal, stagger, layout animation)
- Performance optimization (Core Web Vitals)
- SEO best practices
- Git / GitHub workflow

---

## 4. Full-Stack Developer — Back-end Focus (นักพัฒนาส่วนหลัง)

**JD:**
- พัฒนา CMS ด้วย Strapi (Headless CMS) หรือ Custom Admin
- ออกแบบ Database Schema (MariaDB) สำหรับ 5 Content Types
- พัฒนา RESTful API
- Implement RBAC (Admin/Editor/Viewer)
- ตั้งค่า Elasticsearch สำหรับ Thai Full-Text Search
- Data Migration จากระบบเดิม (7,868 records)

**JS:**
- ปริญญาตรี สาขา Computer Science, IT
- ประสบการณ์ 5+ ปี ในงาน Back-end Development
- เคยทำ Data Migration project อย่างน้อย 1 โครงการ
- เข้าใจ PDPA / Data Privacy

**Skills:**
- Node.js / TypeScript
- Strapi CMS (หรือ Headless CMS อื่น)
- MariaDB / PostgreSQL (Schema design, Query optimization)
- Elasticsearch (Thai tokenizer, ICU plugin)
- REST API design
- Docker / Docker Compose
- Data migration scripting

---

## 5. DevOps / Infrastructure Engineer (วิศวกรระบบ)

**JD:**
- ตั้งค่า Cloud Server (Thailand-based) ตาม TOR spec (8 cores, 16GB, 500GB SSD)
- Deploy ระบบด้วย Docker Compose
- ตั้งค่า Nginx reverse proxy, SSL, IPv6
- ระบบ Backup (Daily/Weekly/Monthly)
- Monitoring 24/7 + Alert system
- Security hardening (Firewall, WAF, OWASP testing)

**JS:**
- ปริญญาตรี สาขา Computer Engineering, IT
- ประสบการณ์ 5+ ปี ในงาน DevOps / System Administration
- มีประสบการณ์กับ Cloud provider ในไทย
- เข้าใจ OWASP Top 10

**Skills:**
- Docker / Docker Compose
- Nginx configuration
- SSL/TLS, IPv6
- Linux server administration
- Monitoring (Grafana, Prometheus, or UptimeRobot)
- Backup automation (cron, shell scripting)
- Security: Firewall (UFW/iptables), fail2ban, WAF
- CI/CD (GitHub Actions)

---

## 6. QA / Tester (ผู้ทดสอบระบบ)

**JD:**
- เขียน Test Plan และ Test Cases ทุกหน้า
- ทดสอบ Functional, Cross-browser, Mobile responsive
- ทดสอบ WCAG 2.1 AA compliance (Accessibility audit)
- ทดสอบ Performance (Lighthouse, Core Web Vitals)
- ทดสอบ Security (OWASP basic scanning)
- จัดทำ UAT documentation

**JS:**
- ปริญญาตรี สาขา IT หรือเทียบเท่า
- ประสบการณ์ 3+ ปี ในงาน QA/Testing
- เข้าใจ WCAG 2.1 AA testing methodology
- ISTQB certification จะพิจารณาเป็นพิเศษ

**Skills:**
- Manual testing (Functional, Regression, UAT)
- Accessibility testing (axe, Lighthouse, screen reader)
- Performance testing (Lighthouse, WebPageTest)
- Cross-browser testing (Chrome, Safari, Firefox, Edge)
- Mobile testing (iOS Safari, Android Chrome)
- Bug tracking (Jira, GitHub Issues)

---

## 7. Content Strategist / Data Specialist (ผู้เชี่ยวชาญข้อมูล)

**JD:**
- วางแผนโครงสร้างข้อมูล 5 หมวด (Films, Series, Personnel, Companies, Archives)
- ตรวจสอบความถูกต้องของ Data Migration (7,868 records)
- เขียน Content สำหรับหน้าต่างๆ (About, Policy, Incentive)
- วางแผน SEO Content Strategy
- จัดทำ Mock Data สำหรับ Demo
- ประสานงานกับกรมส่งเสริมวัฒนธรรมเรื่องข้อมูล

**JS:**
- ปริญญาตรี สาขา นิเทศศาสตร์, สารสนเทศศาสตร์, ภาพยนตร์, หรือเทียบเท่า
- ประสบการณ์ 3+ ปี ในงาน Content Management / Data
- มีความรู้เรื่องอุตสาหกรรมภาพยนตร์ไทย

**Skills:**
- Content management & taxonomy
- Data validation & quality assurance
- SEO copywriting (Thai/English)
- Film industry knowledge
- Excel / Google Sheets (data preparation)
- Basic SQL (สำหรับตรวจสอบ data)

## 8. Campaign Designer / Marketing (นักออกแบบแคมเปญ)

**JD:**
- ออกแบบและพัฒนา 3 Campaign (DNA Quiz, Grand Opening, Youth Challenge)
- สร้าง Share Card templates และ Visual assets
- ประสานงาน Influencer/Creator partnerships
- วางแผน Social Media content calendar (12 เดือน)
- วัดผล Campaign KPIs ด้วย GA4, Meta Business Suite

**JS:**
- ปริญญาตรี สาขา Marketing, Communications, Digital Media
- ประสบการณ์ 3+ ปี ในงาน Digital Campaign
- มีผลงาน viral campaign ที่ทำ 100K+ engagement

**Skills:**
- Social media campaign planning
- Visual design (Canva/Figma for campaign assets)
- Influencer management
- GA4 / Meta Analytics
- Community management
- Copywriting (Thai — catchy, viral-worthy)
- TikTok/Instagram content strategy
